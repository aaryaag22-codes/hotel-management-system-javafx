# hotel-management-system-javafx

A JavaFX-based Hotel Management System that streamlines room booking, customer registration, check-in/check-out, food ordering, billing, and room management. Uses Java Serialization and Deserialization for data persistence and includes SMS reminders for customer engagement.

## Features
- Customer Registration
- Room Booking
- Check-in / Check-out
- Room Availability Tracking
- Food Ordering
- Bill Generation and Receipt Printing
- Add New Rooms
- SMS Reminders

## Technologies Used
- Java
- JavaFX
- Maven
- Serialization & Deserialization
